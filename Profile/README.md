# Profile

"profile별 yml 파일 설정정보 및 Bean 다르게 설정하기"라는 주제로 블로그 글을 작성하며 실습한 코드입니다.



application.yml 파일에서 profile별 환경을 다르게 설정합니다.

`@Profile`를 통해 profile별 활성화 시킬 빈을 다르게 설정합니다.

`@ActiveProfiles`를 통해서, 테스트 코드에서 활성화시킬 profile을 지정합니다.
